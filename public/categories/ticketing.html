Cara Ticketing Helpdesk

  • Buka halaman https://helpdesk.siloamhospitals.com/ pada browser
  • Masukkan username dan password sesuai dengan login captive internet
  • Jika tidak memiliki login internet, bisa menggunakan login internet head department
  • Untuk jenis domain, pastikan LDAP (dibawah kolom password)
  • Klik logo ticket (di pojok kanan atas sebelah icon search) untuk melakukan new request
  • Sesuaikan dengan masalah yang terjadi. Berikut ini beberapa masalah yang sering muncul
  • Jika ada masalah pada PC, Printer, Telefon Cisco, Internet, pilih menu “Incident WTC”.
  • Jika ada masalah pada tv antrian dokter tidak berbunyi, nama dokter tidak muncul, pilih menu “Incident MySiloam”
  • Jika ada masalah lupa password EMR, Hope, MySiloam, silahkan pilih menu Service Catalog, cari menu reset password sesuai masalah atau bisa ketik reset password di kolom pencarian
  • Pilih “Modfiy akses role EMR” jika ingin menambah role EMR
  • Create new user EMR untuk membuat EMR bagi karyawan baru
  • Jika untuk menambah role EMR lebih dari satu, silahkan klik “Create new user Hope”
  • Setelah membuat ticket, klik menu request di tab kiri atas
  • Pastikan untuk mengingat nomor ticket yang baru dibuat
  • Check status ticket. Jika status sudah resolved, klik subject/nama ticket
  • Kemudian pilih resolution untuk mendapat jawaban dari issue

    • (Jika setelah mencoba langkah-langkah di atas masalah masih persisten, hubungi departemen IT atau dukungan teknis untuk bantuan lanjutan)
